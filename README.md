# 🕒 Time Widget

یک ویجت ساعت دسکتاپ مینیمال، شیک و کاربردی برای ویندوز که با **PyQt6** توسعه یافته است. این برنامه قابلیت نمایش زمان و تاریخ شمسی/میلادی شهرهای مختلف را به صورت همزمان دارد.

## ✨ قابلیت‌ها
- نمایش همزمان ساعت و تاریخ چند شهر مختلف با پشتیبانی از تاریخ شمسی و میلادی.
- قابلیت دسکتاپ ویجت (بدون فریم، جابه‌جایی راحت و تغییر میزان شفافیت).
- بخش زمان‌سنج (Timer/Stopwatch) داخلی.
- تنظیم خودکار برای اجرا در زمان بالا آمدن ویندوز (Startup) قابل مدیریت از بخش تنظیمات.
- ذخیره‌سازی استاندارد تنظیمات در پوشه `AppData`.

## 🚀 دانلود و نصب (نسخه آماده)
برای دانلود آخرین نسخه نصب‌کننده ویندوز، بدون نیاز به نصب پایتون، به بخش **[Releases](https://github.com/YOUR_USERNAME/time-widget/releases)** مراجعه کنید و فایل `TimeWidgetSetup.exe` را دانلود کنید.

## 🛠️ نحوه اجرا از طریق کد منبع
اگر می‌خواهید پروژه را خودتان اجرا یا توسعه دهید:

1. پروژه را کلون کنید:
   ```bash
   git clone [https://github.com/YOUR_USERNAME/time-widget.git](https://github.com/YOUR_USERNAME/time-widget.git)

   ```
2. کتابخانه‌های مورد نیاز را نصب کنید:
   ```bash
   pip install -r requirements.txt
   ```
3. برنامه را اجرا کنید:
   ```bash
   python time_screen_widget.pyw
   ```
